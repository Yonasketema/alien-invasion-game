# alien-invasion-game
Alien Invasion from "Python Crash Course " by Eric Matthes.

1. clone the repo
   ```sh
   https://github.com/Yonasketema/alien-invasion-game.git
   ```
2. make .venv dir
   ```sh
   mkdir .venv
   ```
3. Creating virtual environment
   ```sh
   pipenv shell
   ```
4. install packages
   ```sh
   pipenv install
   ```
5. start client
   ```sh
    python3 alien_invasion.py
   ```
